# Stack Overflow Developer Survey — EDA

Дослідницький аналіз великого датасету (49 000+ респондентів)  
з результатами опитування розробників Stack Overflow.

## Що зроблено
- Аналіз повноти даних, робота з пропущеними значеннями, інтерсекція множин
- Розрахунок мір центральної тенденції (mean, median, mode) для досвіду роботи
- Групування по країнах та вікових категоріях, розрахунок медіанної компенсації Python-розробників
- Перцентильний аналіз зарплат, багатокритеріальна фільтрація

## Технології
Python · Pandas · NumPy · Jupyter Notebook

## Датасет
[Stack Overflow Developer Survey 2023](https://survey.stackoverflow.co/2023/) — публічний датасет,  
49 000+ респондентів з усього світу.

## Ключові висновки

**Досвід роботи:**  
Середній досвід (13.37 років) вищий за медіану (10 років) — це свідчить про наявність респондентів з дуже великим досвідом. Медіана і мода співпадають (10 років), тобто найбільш типовий досвід серед розробників — 10 років.

**Python у розрізі вікових груп:**  
Python найпопулярніший серед наймолодших розробників (18–24 роки) — 40%. У всіх інших вікових групах показник стабільно тримається на рівні 36–39%, і лише серед респондентів 65+ трохи нижчий — 31.63%. Python є популярним інструментом незалежно від віку.

**Зарплати Python-розробників по країнах:**  
Рівень компенсації суттєво відрізняється між країнами. У деяких країнах середнє значення значно перевищує медіану — це означає, що невелика кількість респондентів з дуже високими доходами впливає на середнє. Медіана є надійнішим показником типового доходу.

**Високооплачувана віддалена робота:**  
Серед працівників з зарплатою вище 75-го перцентиля, що працюють віддалено, домінує Software Development. Також популярні Fintech, Healthcare та Internet/IT. Технологічні індустрії пропонують найкращі можливості для remote-роботи з високою компенсацією.
